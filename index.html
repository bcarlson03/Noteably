What is Noteably?

A program made in 36 hours during the HackKU23 hackathon that generates educational notes and a transcript from any audio file. The project won "Best Use of Google Cloud". It was created by Michael Stang, Holden Vail, Logan Smith, and Blake Carlson. The theme for the competition was accessability, so Noteably was designed to be a tool for people who are otherwise unable to take notes on their own to have access to academic studying tools.

Technologies used

  • Python
  • Google Cloud Speech-to-Text API
  • Google Cloud Storage API
  • OpenAI API

How it works

Noteably functions as a program that turns various audio files into condensed summaries. The program requires multiple steps in order to turn the raw audio into notes. First the audio file is processed by the Google Cloud speech-to-text API to create a transcript of the original audio file. Next, the transcript is given to a trained large language model through the OpenAI API. The model determines the subject of the audio file and what from the transcript is most important. The model then outputs what it determines to be an accurate and comprehensive summary of the orignal audio source. This output is taken and run through a python script which takes the summarized notes and generates a well-formatted pdf containing the key infomation to be used as notes or a summary.

Takeaways

This was the first hackathon that any of us had competed in. There are several key takeaways and lessons that we learned from our experience.

  • Programming with others as a team is vastly different from working on a project by yourself
  • Splitting up work according to each team members strengths greatly increased productivity
  • Focusing on getting a minimum viable product working is all you can expect to have time for
